Form Builder:

Create and manage multiple simple or complex forms for your website. 
The form builder package supports a wide range of contact form elements such as text field, email field, textarea, dropdown list, radio button, checkbox, date picker, CAPTCHA, file uploader, etc. 
Form Builder is developed by Reza Rad.

Features:
- Full control on form
- Multiple simple or complex forms
- Support the form elements text, email, text area, dropdown, checkbox, radiobutton, date, number, password, hidden, CAPTCHA and file upload.
- Email and/or save form data
- Advanced data validation
- Custom success message and redirection after submission feature
- Advanced CAPTCHA security system with customisation feature
- Style ID and Class integration option with form elements
- Option to add “*” (star symbol) for mandatory fields
- Advanced form customisation feature
